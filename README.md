# Windows-Privilege-Escalation
Privilege Escalation Techniques in Windows Environments

## Enumeration
```
hostname

whoami

whoami /priv

echo %username%

systeminfo

net users

net users <username>

net share

ipconfig /all

arp -a

route print

netsh firewall show

netsh advfirewall show currentprofile

netsh advfirewall show allprofiles

netstat -ano
```
